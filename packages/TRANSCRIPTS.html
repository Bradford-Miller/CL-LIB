(tassert who debug-level test-form &optional places string &rest args)   function

TASSERT who debug-level test-form [({place}*) [string {arg}*]] If the test-form evals to NIL, then either signal an error or log an error, depending on the value of debug-level: if current tracing is such that we would normally trace debug-level, assert instead. If tracing is such that we would ignore, log instead.
